

📈 経済タイムアタック (Economy Time Attack)
~ 投資と国家運営のシナジーを最大化せよ ~
Economy Time Attack は、限られた時間内で資産目標の達成を目指す、スリリングな経済シミュレーションゲームです。株価の変動、人口の増減、資源の生産という3つの要素を構造的に制御し、最適な投資戦略を構築します。

🌟 プロジェクトの概要
このゲームは、単なる「安く買って高く売る」だけの株取引ゲームではありません。
資産を増やすために「発電所」を建設して資源を確保し、それによって「人口（＝市場の買い圧力）」を維持・増大させるという、マクロ経済的なフィードバックループを設計しています。プレイヤーは「投資家」であると同時に「国家の運営者」としての判断を迫られます。

✨ 主なゲームシステム
ダイナミック・マーケット・エンジン:

ランダムウォークに「人口ボーナス」と「イベント倍率」を掛け合わせた独自の株価アルゴリズム。

技術革新、暗黒の月曜日、経済バブルといった突発イベントが市場を激変させます。

国家運営シミュレーション:

人口は時間とともに増加しますが、維持には「資源（Resource）」が必要です。

資源が枯渇すると人口が激減し、ゲームオーバーに直結するシビアな資源管理。

インフラ投資:

資金を投じて発電所を建設。指数関数的に上昇する建設コストと、それに見合う生産力のトレードオフを計算。

ハイスコア永続化:

Easy/Hardの2モードに対応し、ベストタイムをローカルファイルに保存。

🛠 技術スタック
Framework: OpenSiv3D (C++20)

Design Patterns:

State Pattern: GameState によるシーン管理

Emitter Pattern: ParticleManager によるエフェクト制御

UI/UX: MSDFフォントによる高精細なテキスト描画、リアルタイム・ライングラフ

🚀 実行方法
OpenSiv3D の開発環境（Visual Studio / Xcodeなど）を用意します。

Main.cpp にソースを配置し、ビルド・実行してください。

🕹 攻略のヒント
序盤: 発電所に投資し、人口を安定させることが急務です。人口が少ない状態では株価の伸びも期待できません。

中盤: バブルイベントが発生した瞬間を見逃さず、全力で株を買い増し、ピークで売り抜ける「出口戦略」がタイム短縮の鍵です。

終盤: 目標額が近づいたら、インフラ投資を止め、全資金を市場へ投入して総資産をブーストさせましょう。
